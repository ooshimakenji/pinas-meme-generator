# Pinas Meme Generator

Gerador de memes com React. Escolha uma imagem, adicione texto em cima e embaixo, e crie seu meme.

## Funcionalidades

- Busca de templates de memes
- Texto customizável (topo e base)
- Geração de memes em tempo real
- Interface simples e divertida

## Stack

- **React 19** + Vite
- JSX components (Header, Main)

## Instalação

```bash
git clone https://github.com/ooshimakenji/pinas-meme-generator.git
cd pinas-meme-generator
npm install
npm start
```
